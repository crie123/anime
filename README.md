# Anime Collection

Кратко: локальное настольное приложение на Kivy для хранения каталога аниме (постеры, скриншоты, теги, описание).

Требования:
- Python 3.8+ (рекомендуется 3.10–3.13)
- kivy>=2.0.0
- pillow

Установка (строго):
1. Создать виртуальное окружение: python -m venv venv
2. Активировать: venv\Scripts\activate
3. Установить зависимости: pip install kivy pillow

Запуск (Windows):
- python main.py

Файловая структура (основное):
- main.py — точка входа, UI и логика
- anime.kv — правила интерфейса Kivy
- database.py — простая обёртка для sqlite
- popups.py — вспомогательные диалоги/попапы
- utils.py — утилиты (миниатюры, копии файлов и т.д.)
- thumbnails/ — генерируемые миниатюры (не хранить в репозитории)
- copies/ — локальные копии исходных изображений (не хранить в репозитории)
- anime.db — локальная БД (не в репозитории)

Важные примечания:
- Файлы thumbnails/ и copies/ генерируются автоматически и должны игнорироваться (.gitignore включён).
- Приложение использует системные вызовы Windows для перетаскивания окна; тестирование и запуск на других ОС не гарантируются.

Поддержка: issues в репозитории.
